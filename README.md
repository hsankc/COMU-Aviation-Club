# 🦅 ÇOMÜ Havacılık Kulübü (CHK) Web Portalı

Çanakkale Onsekiz Mart Üniversitesi Havacılık Kulübü için geliştirilmiş, modern, interaktif ve premium kullanıcı deneyimi sunan resmi web sitesi.

![ÇOMÜ Havacılık](public/club-logo.png)

## ✨ Öne Çıkan Özellikler

### 🪐 Sinematik Hero Bölümü
- **Warp Speed (Hyperspace)**: Mobil cihazlarda derin uzay yolculuğu hissi veren akıcı animasyonlar.
- **Dinamik F-16**: Fotorealistik jet görselleri, süzülme efektleri ve afterburner detayları.
- **Super Glow Logo**: Kulüp logosu etrafında üç katmanlı, periyodik olarak parıldayan (glint) efektlerle zenginleştirilmiş marka kimliği.
- **Atatürk İmzası**: Vektörel ve asil bir duruşla yerleştirilmiş Atatürk imzası ve sözü.

### 👥 Profesyonel Yönetim Ekibi
- **Liderlik Hiyerarşisi**: Akademik danışman ve başkanın öne çıkarıldığı, dengeli ve şık bir ekip yerleşimi.
- **Görsel Denge**: Fotoğrafların JIMP ile optimize edildiği ve boyutlarının altın orana göre ayarlandığı profil kartları.
- **Responsive Dizilim**: Desktopta **4'lü**, mobilde **3'lü** profesyonel grid sistemi.

### 📚 Akademi & Eğitimler
- **Akordeon (Expandable) Kartlar**: Mobilde uzun açıklamaları gizleyerek temiz bir görüntü sunan, tıklandığında yumuşak animasyonla açılan eğitim detayları.
- **Entegre Başvuru Sistemi**: Simülasyon merkezi, UAVET ve İHA eğitimleri için hazır form ve WhatsApp bağlantıları.

### 🛠️ Teknik Altyapı
- **Supabase**: Etkinlik, galeri ve içerik yönetimi için bulut tabanlı backend entegrasyonu.
- **Framer Motion**: Tüm sitede kullanılan profesyonel scroll-reveal, stagger ve hover animasyonları.
- **Responsive Design**: Masaüstünden mobile her cihazda kusursuz (Pixel-Perfect) ve performanslı görünüm.
- **İLİON7 Branding**: Geliştirici kimliğini yansıtan şık ve interaktif marka yerleşimleri.

## 🚀 Başlarken

### Gereksinimler
- **Node.js**: v18 veya üzeri
- **NPM** veya **Yarn**

### Kurulum

1. Depoyu klonlayın:
```bash
git clone https://github.com/hsankc/CHK.git
cd CHK
```

2. Bağımlılıkları yükleyin:
```bash
npm install
```

3. Çevre değişkenlerini (`.env`) ayarlayın:
Root dizinine bir `.env` dosyası oluşturun ve Supabase bilgilerinizi ekleyin:
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

4. Geliştirme sunucusunu çalıştırın:
```bash
npm run dev
```
